전체 선택자
*{color:red;} 모든 요소에 red색상 적용

태그선택자
p{color: red;} 해당 태그를 가진 모든 요소에 red색상 적용

클래스 선택자
.className{ color:red;} 해당 클래스를 가진 모든 요소에 red색상 적용

ID 선택자
#id{color:red;} 주어진 id속상값을 가진 요소를 선택해 red색상적용
id: 고유한 식별자 역할을 하는 전역속성 (단 하나만 존재)

그룹 선택자
h1,p, div{color:red;} 문서내 해당 태그들을 가진 요소들에 red색상 적용

선택자 우선순위 
기본적으로 나중에 작성된 스타일이 적용
선택자가 다르지만 요소가 곂치는 경우
ID선택자 > 클래스 선택자 > 태그 선택자
